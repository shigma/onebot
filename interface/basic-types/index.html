    基本数据类型

    整数

    int64、uint64、int32、uint32、int16、uint16、int8、uint8

    浮点数

    float64

    字符串

    string

    字节序列

    bytes

    在 JSON 中表示为 Base64 编码的字符串，MessagePack 中表示为 bin 格式的字节数组。

    字典/映射/对象

    map[<key_type>]<value_type>

    其中，<key_type> 为键类型，<value_type> 为值类型。

    动作响应数据

    resp[`<action_name>`]

    其中，<action_name> 为动作名。
